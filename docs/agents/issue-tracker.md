# Issue Tracker（GitHub）

本仓库的问题跟踪系统为 GitHub Issues，仓库地址为：

- `jksuan/polymarket-seer`

## Agent 执行约定

- 创建 issue：使用 `gh issue create`
- 查看 issue：使用 `gh issue view <number>`
- 列出 issue：使用 `gh issue list`
- 评论 issue：使用 `gh issue comment <number> --body "..."`

## 使用原则

- 任务拆分（`to-issues`）默认写入 GitHub Issues
- PRD 产物（`to-prd`）默认作为 GitHub issue 发布
- triage 流程对标签的读写以 `docs/agents/triage-labels.md` 为准
