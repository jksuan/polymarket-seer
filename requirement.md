# 体育预测 H5 应用前端设计需求文档 (PRD)

## 1. 项目视觉风格定位 (Visual Style)
* **核心理念**：去交易化、直觉化、高颜值、社交感。
* **设计风格**：**新野兽主义 (Neobrutalism)**。使用粗边框、高饱和度撞色（如荧光绿 #ADFF2F）、硬核阴影。
* **适配目标**：移动端 H5（特别优化 X/Twitter 内置浏览器体验）。

---

## 2. 页面布局规范 (Frontend Layout)

### 2.1 Header (页眉)
* **功能布局**：固定在顶部 (Sticky)，采用毛玻璃背景。
* **左侧**：品牌动态 Logo（点击回首页）。
* **右侧**：
    * **未登录**：显示“一键开赛”按钮（支持社交账号一键登录）。
    * **已登录**：显示用户头像 + $实时余额。

### 2.2 Navigation (频道导航栏)
* **交互形式**：横向滚动选项卡 (Horizontal Scroll Tabs)。
* **频道序列**：`实时` | `NBA` | `欧冠` | `英超` | `意甲` | `西甲` | `德甲` | `网球`。
* **视觉细节**：文字上方配极简体育图标；选中项触发“呼吸灯”高亮效果。

### 3.3 Main Content (主体内容 - 预测卡片)
* **禁止展示**：K 线图、订单簿、买卖深度。
* **卡片设计要素**：
    * **头部**：联赛名称 + 比赛倒计时（营造紧迫感）。
    * **中段 (对决区)**：主队图标 vs 客队图标。背景使用双色渐变（根据球队主色调）。
    * **数据转化**：通过 Polymarket API 的 `price` 实时转化为：
        * **胜率百分比** (例: 70% 胜率)
        * **赔率格式** (例: 1.45x 收益)
    * **操作区**：两个超大彩色按钮。文案：“支持 [队名]”。
    * **人气指数**：底部显示“🔥 1.2k 人已参与预测”。

### 3.4 Footer (底部导航栏)
* **五个 Tab 菜单**：
    1.  **首页 (Discover)**：热门赛事发现流。
    2.  **搜索 (Search)**：球队/球星快速查找。
    3.  **挑战 (Quest)**：新手任务、每日签到、积分兑换（留存核心）。
    4.  **英雄榜 (Leaderboard)**：收益排行，支持查看“大神”预测。
    5.  **更多 (More)**：个人历史记录、钱包管理。

---

## 4. 技术与社交对接 (Technical & Social)
* **交易逻辑**：前端默认执行“市价成交 (Market Order)”，屏蔽复杂限价设置。
* **社交分享**：
    * 一键生成“战报海报”（Canvas 生成）。
    * 海报包含：预测内容 + 实时赔率 + 比赛视觉图 + 专属二维码。
* **性能要求**：极速加载，使用骨架屏 (Skeleton Screen) 减少白屏焦虑。
